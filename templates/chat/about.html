Anonymous Chat

This is Anonymous Chat created using Django Channels to implement a simple WebSocket-based chat server.

How to Use:

To use this app you can either create a random named room using the button below or add /room-name at the end of this url to create a room. Share the room's url to invite others for a chat where you can choose any name you want. It would give you the power to speak without revealing you identity. Invite your partner by sharing this link via apps like sarahah. Remember you have to change the . in the link to [dot] other wise you won't be able to share this in many sites. You can see the code on GitHub, or try the app:
